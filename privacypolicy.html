In this page we inform the visitors about the privacy policy and terms of the application Automata Theory App. By installing our application you agree to the terms presented in this page and accept to know the privacy policy regarding the collection, use, and disclosure of personal information when using Automata Theory App.

Terms

Automata Theory App is an application that provides free access to a brief summary of subjects from automata theory and formal languages. It also provides access to solutions of selected exercises under a cost. In order to access paid solutions, the user needs to buy Automata Theory App Coins for Knowledge (ATACK) and, for that purpose, Automata Theory App uses Internet access so the user is able to make payments and receive confirmation of payments for the application to grant access to paid solutions. The user is the only responsible person for any misuse of the application.

What information does Automata Theory App collect?

Automata Theory App collects information about payments done in order to update information about available and used coins. It also stores information about exercises bought by the user in order to grant access to their solutions. Automata Theory App collects anonymous information on sections of the app that are accessed by the users in order to use that information to improve the application. No other kind of information is collected by Automata Theory App.

How Automata Theory App uses the information?

Information about payments, available coins and exercises bought are used in order to provide the user the proper access to selected solutions. Information about the usage of the app is used in order to detect possible improvements to implement in future releases.

What information does Automata Theory App share?

None.

Changes to this Privacy Policy and Terms

Updates to this privacy policy will be made with new releases of Automata Theory App. It is advised to check this page frequently to check new changes. Changes of this Privacy Policy are notified on this page and are effective immediately.
